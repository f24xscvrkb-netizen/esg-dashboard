    Methodology

    This dashboard is intended as an exploratory tool rather than a definitive ranking, highlighting how ESG scores and controversy data can be jointly interpreted.

    How to read the chart

    • X-axis: ESG Score (higher values indicate better performance according to the selected metric).
    • Y-axis: Number of controversies (higher values indicate more reported events).
    • Quadrants: The ideal position is the bottom-right quadrant (high ESG score, low number of controversies).

    Limitations (important)

    • ESG metrics are not fully standardized; different rating providers may produce different results.
    • Controversy data are influenced by media coverage, disclosure practices, and event definitions.
    • A professional analysis would require controls for firm size, sector, geography, and the chosen time window.
